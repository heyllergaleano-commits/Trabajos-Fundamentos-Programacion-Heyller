# Quiz Semana 3 – Pilares POO y Principios SOLID 🧩

## Descripción

Este proyecto corresponde al **Quiz de la Semana 3** de la asignatura **Fundamentos de Programación**.

El programa implementa los conceptos de **Programación Orientada a Objetos (POO)** y aplica los principios **SOLID**, especialmente:

- SRP (Single Responsibility Principle)
- OCP (Open/Closed Principle)

El sistema permite leer la información de empleados desde un archivo de Excel, crear los objetos correspondientes según el tipo de empleado y calcular automáticamente su pago.

---

## Tecnologías utilizadas

- Python 3
- Programación Orientada a Objetos (POO)
- Pandas
- OpenPyXL

---

## Archivos del proyecto

- `quiz_nomina.py` → Programa principal.
- `empleados.xlsx` → Archivo con la información de los empleados.
- `requirements.txt` → Librerías necesarias.
- `README.md` → Documentación del proyecto.

---

## Instalación

Instalar las dependencias:

```bash
pip install -r requirements.txt
```

Ejecutar el programa:

```bash
python quiz_nomina.py
```

---

## Funcionalidades

- Lectura de empleados desde Excel.
- Uso de herencia y polimorfismo.
- Validación de datos.
- Manejo de excepciones.
- Cálculo de nómina.
- Identificación del empleado con mayor salario.
